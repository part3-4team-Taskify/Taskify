# Installation

1. Clone the repository

```bash
git clone https://github.com/part3-4team-Taskify
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm start dev
```

4. Open the project in your browser

```bash
http://localhost:3000
```
