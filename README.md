# Package Structure

```
taskify
├─ public
│  ├─ svgs                  # 아이콘 리소스
│  └─ images                # 이미지 리소스
├─ src
│  ├─ api                   # API 사용을 위한 세팅
│  ├─ components            # 주요 컴포넌트
│  ├─ constants             #
│  ├─ hocks                 # 인증, 모달 컨텍스트 프로바이더
│  ├─ pages                 # 커스텀 훅
│  ├─ shared                #
|  ├─ store                 #
├─ styles
│  └─ globals.css           # 폰트
└─ types                    # 스타일
```

# Installation

1. Clone the repository

```bash
git clone https://github.com/part3-4team-Taskify
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm start dev
```

4. Open the project in your browser

```bash
http://localhost:3000
```
